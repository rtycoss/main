3D Semantic Segmentation

The 3D segmentation project is aimed at developing and implementing networks for use in medical image segmentation on three dimensional data such as MRI and CT scan volumes. The project looks primarily at developing 3D transformer networks for segmentation with a number of other models available for comparison. It concerns mostly methods which provide performance improvements in the unbalanced multi-class scenario as this tends to be the underlying problem for many medical image segmentation use-cases. Model development is currently done on publically available CT and MRI data with a view to application on clinical data from these domains.
